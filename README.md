# 🎮 VR Squid Game – Weapon Interaction (Unity)

## 🧠 Overview

🔴 VR-based weapon interaction prototype inspired by Squid Game
🟢 Focus on immersive gameplay and physics-based interactions
🔵 Built as a learning/demo project in Unity

## 🚀 Features

🔴 **VR Interaction System**
🟢 Grab, hold, and manipulate objects using VR controllers
🔵 Realistic physics-based interactions

🔴 **Weapon Handling**
🟢 Pick up and use different weapon types
🔵 Basic shooting and throwing mechanics

🔴 **Target Practice Environment**
🟢 Dummy enemies / training targets
🔵 Hit detection and feedback

🔴 **Game UI Elements**
🟢 Timer system (“Time Left”)
🔵 State indicators (e.g., Running)

🔴 **3D Arena Setup**
🟢 Simple enclosed environment
🔵 Stylized assets and props

## 🛠️ Built With

🔴 Unity (Tested on Unity 6+)
🟢 XR Interaction Toolkit / Custom VR scripts
🔵 C#

## ⚙️ Getting Started

🔴 Clone the repository
🟢 Open in Unity Hub
🔵 Load: Assets/Scenes/SampleScene.unity
🔴 Connect VR headset (Oculus / OpenXR)
🟢 Press Play ▶️

## 🎮 Controls

🔴 Grab Weapon – Trigger / Grip
🟢 Shoot / Use – Primary button
🔵 Move – Joystick / Thumbstick
